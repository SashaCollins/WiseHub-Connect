## Frontend (Vuejs)
### Requirements
- nodejs v15.10.0 & npm v7.6.1

#### Project setup
```
npm install
```

##### Compiles and hot-reloads for development
```
npm run serve
```

##### Compiles and minifies for production
```
npm run build
```
